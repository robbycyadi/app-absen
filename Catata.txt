Oke, saya bantu langkah demi langkah:
Langkah 1: Ambil Firebase Service Account Key
Buka link ini di browser:
https://console.firebase.google.com/project/app-absensi-296ee/settings/serviceaccounts/adminsdk
Klik tombol "Generate new private key" (warna biru) → klik "Generate key" → akan terdownload file .json
Langkah 2: Masukin ke GitHub Secret
Buka link ini di browser:
https://github.com/USERNAME/App-Absen/settings/secrets/actions
(ganti USERNAME dengan username GitHub kamu)
1. Klik tombol "New repository secret" (tombol hijau)
2. Di Name tulis: FIREBASE_SERVICE_ACCOUNT_APP_ABSENSI_296EE
3. Di Secret paste isi file .json yang tadi (buka file .json pakai text editor, copy semua, paste)
4. Klik "Add secret"
Selesai! Setelah itu tiap push code ke branch main, GitHub otomatis build web dan deploy ke Firebase.
